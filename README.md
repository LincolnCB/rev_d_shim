# Shim Amplifier System, Rev. D

This repo is the source code for the Rev. D version of our shim amplifier system. It is primarily written by me, Lincoln Craven-Brightman, and based on the work of Pavel Demin, who provided a great starting point with his repo for Zynq-7000 development.

To use this repo, you will need the following repos. As of 2026-06-22, they have been archived on Zenodo, and the DOIs are provided below.
- Our build environment repo, [Zynq Toolbox](https://github.com/LincolnCB/zynq_toolbox) ([https://github.com/LincolnCB/zynq_toolbox](https://github.com/LincolnCB/zynq_toolbox) [![DOI](https://zenodo.org/badge/846674502.svg)](https://doi.org/10.5281/zenodo.20802348)), which is an extension of Pavel's original [Red Pitay Notes](https://github.com/pavel-demin/red-pitaya-notes) ([https://github.com/pavel-demin/red-pitaya-notes](https://github.com/pavel-demin/red-pitaya-notes)) repo.
- The hardware design files for the Rev. D version of our shim amplifier system, which have been created by Don Straney and are available in our [hardware repo](https://github.com/stockmann-lab/shim_amp_hardware_linear/) ([https://github.com/stockmann-lab/shim_amp_hardware_linear/](https://github.com/stockmann-lab/shim_amp_hardware_linear/) [![DOI](https://zenodo.org/badge/587958969.svg)](https://doi.org/10.5281/zenodo.20802309))
